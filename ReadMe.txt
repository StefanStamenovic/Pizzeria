Na pocetku Index metode Home kontrolera se pozivaju metode DeleteAllData i Initialize, po potrebi otkomentarizati/zakomentarisati radi reseta baze iz tekstualnih fajlova. U folderu Backup je takodje data slika baze kreirana putem mongodump modula, kao rezerva.

Koriscene tehnologije: 
ASP.NET MVC5, Bootstrap, JS, jQuery, MongoDB
